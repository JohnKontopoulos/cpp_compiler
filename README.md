# CPP Compiler

Μεταγλωττιστής για τη γλώσσα CPP (απλοποιημένη C++) που παράγει κώδικα MIPS assembly.

Αναπτύχθηκε στα πλαίσια εργασίας για το μάθημα **Μεταγλωττιστές**.

## Περιγραφή

Ο μεταγλωττιστής υλοποιεί τα εξής στάδια:

1. **Λεκτική Ανάλυση** — Αναγνώριση λεκτικών μονάδων με flex
2. **Συντακτική Ανάλυση** — Ανάλυση γραμματικής με bison
3. **Σημασιολογική Ανάλυση** — Έλεγχος τύπων και κατασκευή ΑΣΔ
4. **Βελτιστοποίηση** — Βελτιστοποιήσεις ενδιάμεσου κώδικα
5. **Παραγωγή Κώδικα** — Παραγωγή MIPS assembly

## Εγκατάσταση

### Απαιτήσεις

- **flex** >= 2.6
- **bison** >= 3.0
- **gcc** >= 9.0
- **make**

### Μεταγλώττιση

```bash
make
```

### Καθαρισμός

```bash
make clean
```

## Χρήση

```bash
# Εκτύπωση MIPS στο stdout
./build/compiler input.cpp

# Αποθήκευση MIPS σε αρχείο
./build/compiler input.cpp output.asm
```

### Έξοδος

- **stdout**: MIPS assembly κώδικας
- **stderr**: Debug πληροφορίες (tokens, ΠΣ, ΑΣΔ, βελτιστοποιήσεις)

## Στάδια Μεταγλώττισης

### Στάδιο 1 — Λεκτικός & Συντακτικός Αναλυτής

- Case-insensitive λέξεις-κλειδιά (εκτός CCONST/SCONST)
- Υποστήριξη σταθερών σε δεκαδική, δεκαεξαδική (0X), οκταδική (0O), δυαδική (0B) μορφή
- Panic mode error recovery (μέχρι 5 σφάλματα)
- Εκτύπωση αριθμού γραμμής και περιεχομένου γραμμής σε σφάλματα
- Επίλυση conflicts με προτεραιότητα/προσεταιριστικότητα τελεστών

### Στάδιο 2 — Σημασιολογικός Αναλυτής

- Κατασκευή Αφηρημένου Συντακτικού Δέντρου (ΑΣΔ)
- Πίνακας Συμβόλων με hash table και διαχείριση εμβελειών
- Έλεγχος τύπων και αυτόματη εισαγωγή cast nodes (int↔float)
- Χώρος Δεδομένων με υπολογισμό offsets
- Υποστήριξη typedef, const, παραμέτρων by value/reference

### Στάδιο 3 — Γεννήτορας MIPS Κώδικα

- Παραγωγή MIPS assembly για όλες τις κατασκευές της γλώσσας
- Διαχείριση stack frame για main και συναρτήσεις
- Είσοδος/Έξοδος μέσω `_printf`/`_scanf` με format strings
- Δέσμευση καταχωρητών $t0-$t7 και $f0-$f14
- Αποθήκευση/επαναφορά caller-saved registers σε κλήσεις

## Βελτιστοποιήσεις

Ο μεταγλωττιστής υλοποιεί τρεις βελτιστοποιήσεις ενδιάμεσου κώδικα:

| #   | Βελτιστοποίηση           | Παράδειγμα              |
| --- | ------------------------ | ----------------------- |
| 1   | Αποτίμηση Σταθερών       | `2+3` → `5`             |
| 3   | Απαλοιφή Άχρηστου Κώδικα | κώδικας μετά `return`   |
| 4   | Διάδοση Αντιγράφου       | `x=5; y=x` → `x=5; y=5` |

## Εργαλεία

- **flex** — Λεκτικός αναλυτής
- **bison** — Συντακτικός αναλυτής LALR(1)
- **gcc** — Μεταγλωττιστής C
- **MARS/SPIM** — Προσομοιωτής MIPS για επαλήθευση
